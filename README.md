
Opzetten project op jouw machine:
 ```
 npm install
 ```
 
Om de de Express server op te starten run:
```
npm run start 
```

of 
```
yarn start 
```

Om je css en js te compilen run:
```
npm run dev 
```

of 
```
yarn dev 
```

Om watch te gebruiken run:
```
npm run watch 
```

of 
```
yarn watch 
```
